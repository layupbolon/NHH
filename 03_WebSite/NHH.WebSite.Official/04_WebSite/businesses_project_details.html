Project profile

占地面积：23.95万平方米
建筑面积：70.16万平方米

绿化率：30%
地理位置
项目位于山东省章丘市明水城区核心地段，北起汇泉路，南至胶济铁路，西至明水大街、党校街。北侧为百脉泉广场和百脉泉公园，周边坐拥百脉泉、清照园等名胜，区位条件十分优越。
交通状况
距离地铁10号线金台夕照站及地铁1号线永安里站约300米
周边有28 路、43 路、120 路、639 路等10 余条公交线路
咨询热线：0459-6036595
项目介绍 图片展示 场地租借及多种经营 广告位招租 我要入驻
全市规模最大、业态最全、档次最高的综合商业中心。旨在为章丘市民乃至中外游客提供高品质的购物、休闲娱乐和配套服务，并成为章丘商业商机换代的超级引擎。七大商业亮点：全称核心商圈定位、规模最大品类最全、休闲娱乐品类众多、全城最大儿童业态、全城婚礼举办地标、餐饮类别丰富多样、民俗文化体验中心。

规划特色

商业业态包括：特色零售、家居、运动、儿童、休闲大型餐饮、KTV、电玩城、影视城以及商业步行街，是集购物、休闲、餐饮、娱乐为一体的一站式大型综合商业体。本项目A地块商业部分沿汇泉路设计主题入口广场及城市公共广场，作为商业地块主要的人流集散区，沿汇泉路设计大型百货及综合娱乐楼，与城市公共空间及人流形成良好的互动。百货及娱乐楼以南侧商业部分上面布置两栋高层住宅。其中，百货及娱乐楼五层，商业街两层，办公20层，住宅17层。
立天唐人将鼎力打造本项目成为具有大型百货、休闲娱乐、商业步行街、办公、高级住宅等功能的大型城市综合体。其中北侧A地块为商业综合用地，B、C、D、G、F、H地块以住宅及底层商业为主。项目总建筑面积约为70万方，其中地上建筑面积约为54万方，地下建筑面积约为16万方。商业部分面积约为20万方，住宅部分面积约为34万平方米。
